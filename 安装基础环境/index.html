快速开始

适用于可访问互联网的服务器，从 YUM 仓库直接安装：

bash
# 默认使用阿里云镜像
./setup_centos7.sh -m online

# 使用清华镜像源
./setup_centos7.sh -m online -r tsinghua

# 试运行（不实际执行）
./setup_centos7.sh -m online -d

适用于无法访问互联网的服务器，需先在有网络的机器上下载 RPM 包：

1

下载 RPM 包（有网络的服务器）

bash
chmod +x download_rpms.sh
./download_rpms.sh
# 生成 rpms_centos7/ 目录
2

拷贝文件到目标服务器

将 setup_centos7.sh 和 rpms_centos7/ 目录拷贝到目标服务器

3

执行安装（目标服务器）

bash
./setup_centos7.sh -m offline -p ./rpms_centos7

批量部署

bash
SERVERS=("192.168.1.10" "192.168.1.11" "192.168.1.12")

for server in "${SERVERS[@]}"; do
    echo "部署到: $server"

    # 拷贝文件
    scp setup_centos7.sh rpms_centos7/ root@${server}:/opt/

    # 执行安装
    ssh root@${server} "cd /opt && ./setup_centos7.sh -m offline -p ./rpms_centos7"
done
yaml
- name: 安装基础环境
  hosts: centos7_servers
  become: yes
  tasks:
    - name: 拷贝安装脚本
      copy:
        src: setup_centos7.sh
        dest: /opt/setup_centos7.sh
        mode: '0755'

    - name: 拷贝 RPM 包
      copy:
        src: rpms_centos7/
        dest: /opt/rpms_centos7/

    - name: 执行安装脚本
      command: /opt/setup_centos7.sh -m offline -p /opt/rpms_centos7
      args:
        creates: /var/log/env_setup/

安全注意事项

⚠
SELinux 和防火墙
脚本会关闭 SELinux 和防火墙。生产环境请按需配置安全策略，不要完全关闭。
⚠
软件源安全
线上安装请确保使用官方或可信镜像源。本地 RPM 包需从可信来源获取。
ⓘ
安装后建议：检查系统配置是否符合安全要求，根据业务需要调整防火墙规则和 SELinux 策略。
